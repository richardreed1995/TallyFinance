# Vista Office Template

A Next.js template for building private office pages with assessment, consultation, and chat functionality.

## Features

- **Assessment System**: Multi-step assessment with progress tracking
- **Consultation Booking**: Appointment scheduling with qualification checks
- **Chat Interface**: Real-time chat functionality
- **Results Pages**: Dynamic result display based on assessment outcomes
- **Responsive Design**: Mobile-first approach with modern UI components
- **Database Integration**: Supabase integration for data persistence

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **Styling**: Tailwind CSS with Radix UI components
- **Database**: Supabase
- **Forms**: React Hook Form with Zod validation
- **Icons**: Lucide React
- **Charts**: Recharts

## Getting Started

1. Clone this repository
2. Install dependencies: `pnpm install`
3. Set up your environment variables (see `.env.example`)
4. Run the development server: `pnpm dev`
5. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

- `app/` - Next.js app router pages and layouts
- `components/` - Reusable UI components
- `lib/` - Utility functions and database configuration
- `scripts/` - Database migration scripts
- `public/` - Static assets

## Customization

This template is designed to be easily customizable for different office or business contexts. Update the content, styling, and functionality to match your specific needs.
